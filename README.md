Movie Search App. 

A modern React application that allows users to search for movies in real-time using the OMDb API. Built with React and Tailwind CSS, it demonstrates API integration, state management, and responsive front-end design.

Features : 

Search for movies by title with live results
Display detailed movie information: title, year, genre, plot, and poster
Responsive and user-friendly interface using Tailwind CSS
Component-based architecture for clean, maintainable code
Integration with the OMDb API to fetch real movie data
